# cmed
A Stata Module for Analysis of Causal Mediation

## About 
`cmed` is a Stata module for implementing the methods described in Causal Mediation Analysis (Wodtke and Zhou Forthcoming, Cambridge University Press). This package is live but also currently under further development. Functionality may change, and there may be bugs or incomplete features. Please report any issues to causalmed (at) gmail (dot) com.
